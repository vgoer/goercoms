# GoerComs
> 学习中开发的组件

## 1. 安装
> npm i goercoms

## 2. 使用

> 组件1 vue3 暗黑主题切换

> 依赖 饿了么ui样式
```js
1. 安装
npm install element-plus --save

2. main.js 使用 暗黑样式
import 'element-plus/theme-chalk/dark/css-vars.css'
```


```js
//  goer 自己的样式
import 'goercoms/style.css' 

// 按需引入到使用的文件
import {GoBtn,GoDark} from 'gocoms'

<GoDark></GoDark>
```
